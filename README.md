# Mentors

Mentors is a social initiative where accomplished professionals become role models to young people to provide free mentorship sessions.

## Pre-requisites

- npm version >= 6.13.4
- node version >= 12.14.1

## Installing Mentors

`npm install`

## Running tests

`npm run test`
